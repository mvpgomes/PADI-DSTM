PADI-DSTM
=========

Para correr a aplicação PADI-DSTM deverá ser executada as seguintes operações :

1 - Correr o projecto MasterServer, que por defeito está a correr no porto 8086

2 - Correr uma ou mais instancias do projecto DataServer, e assim escolher o porto
para cada DataServer.

3 - Correr uma ou mais instâncias do projecto ClientApplicationl onde deverá proceder
os seguintes passos:

3.1 - Clicar no botão Init

3.2 - E assim proceder as operações de forma normal.
